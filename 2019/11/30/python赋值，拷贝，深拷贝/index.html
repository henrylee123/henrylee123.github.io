python 的 copy

我们写写 python 的赋值，copy, deepcopy 三者之间的关系

  1. =（赋值）: 给对象加了一个引用，并没有创建新对象
  2. copy: 有创建息对象，若是对象内引用了别的对象则只创建引用，不深究
  3. deepcopy: 有创建对象，并且是跟源对象一样的对象，且是全新的跟之前的没有任何联系的对象
