linux 常见命令

发表于 2019-11-14

常见命令

  1. cd
    打开文件

  2. ls -l -a
    显示文件

  3. grep
    正则匹配

  4. find
    查找文件

  5. mkdir
    新建文件夹

  6. rm
    删除文件、文件夹

  7. cp
    复制

  8. mv
    剪切

  9. ln -s
    建立连接 -s 为软链接
    软链接在删除原文件后不能访问成功
    硬链接在删除原文件后不受影响

  10. ps -aux -ef
    查看进程信息

  11. top
    查看最进程详细信息

  12. kill
    杀死进程

  13. tar
    解压

  14. chmod
    授权文件
    rwx = 421

  15. chown
    更爱文件拥有用户
    chown -R henrylee:users *

  16. tail
    -f 查看正在改变的日志
    -3000 倒数3000行数据 | grep ‘A’

  17. nohup &

启动后台进程: command &
挂起进程:

  1. cat
    查看文件

  2. vim/vi
    编辑器

  3. netstat
    网络状态

  4. ifconfig
    网络连接信息

内存利用率

  1. top 10 进程内存利用率
    ps auxw|head -1;ps auxw|sort -rn -k5|head -10

  2. 内存利用率
    vmstat

  3. 清除内存
    sudo sh -c ‘echo 1 > /proc/sys/vm/drop_caches’

cpu 利用率

top -bn 1 -i -c
