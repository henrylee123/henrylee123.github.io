linux 常见命令

Posted on 2019-11-14 Edited on 2019-12-04 In 操作系统

复习下 linux 的操作命令

常见命令

  • cd
    打开文件

  • ls -l -a
    显示文件

  • grep
    正则匹配

  • find
    查找文件

  • xargs

作为列标参数传给下个命令执行

  • mkdir
    新建文件夹

  • rm
    删除文件、文件夹

  1. 删除文件夹 -r
  2. 删除大批量文件

  • cp
    复制

  • mv
    剪切

  • ln -s
    建立连接 -s 为软链接
    软链接在删除原文件后不能访问成功
    硬链接在删除原文件后不受影响

  • ps -aux -ef
    查看进程信息

  • top
    查看最进程详细信息

Load Avg 可以判断出系统健康，一般为 1 左右是健康的。
第四行是看内存的使用量

  • kill
    杀死进程

  • tar
    解压

  • chmod
    授权文件
    rwx = 421

  • chown
    更爱文件拥有用户
    chown -R henrylee:users *

  • tail
    -f 查看正在改变的日志
    -3000 倒数3000行数据 | grep ‘A’

  • nohup &

  1. 启动后台进程: command &

输出一样会 print 到屏幕上。

  1. 挂起进程: nohup command
  • 2>1

这是一个奇怪的命令

ls 2>1 输出一个空文件 1
ls xxx 2>1 没有xxx的错误信息输出到了 1
ls xxx 2>&1 输出方式变为标准输出 -> 命令行 print 了出来
ls xxx > out.txt 2>&1 标注输出变为了输出到 out.txt。所以错误输出到了标准输出 out.txt 里面

  • cat
    查看文件

  • vim/vi
    编辑器

  • netstat
    网络状态

  • ifconfig
    网络连接信息

  • df -h

磁盘使用率 100%

udev            2.0G     0  2.0G   0% /dev
tmpfs           395M  716K  394M   1% /run
/dev/sda1        31G   31G     0 100% /
tmpfs           2.0G     0  2.0G   0% /dev/shm
tmpfs           5.0M     0  5.0M   0% /run/lock

  • 内存利用率

  1. top 10 进程内存利用率
    ps auxw|head -1;ps auxw|sort -rn -k5|head -10

  2. 内存利用率
    vmstat

  3. 清除内存
    sudo sh -c ‘echo 1 > /proc/sys/vm/drop_caches’

cpu 利用率

top -bn 1 -i -c
