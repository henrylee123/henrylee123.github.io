buffer cache 的区别

Posted on 2019-11-29

cache

cache 主要作用缓解 cpu 与普通内存的速度差异。把频繁的寻址地址缓存在 cache 能够优化系统性能。

cache 有三级，一级比一级快。

buffer

buffer 一定程度上减少对 i/o 的设备访问次数。 比如，写入一个文件到磁盘中的时候，会先存一部分在 buffer 里，攒到一定量的时候再写回磁盘中。减少对此盘的访问次数
